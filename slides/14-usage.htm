شروع سریع

کافی است فایل CSS را مستقیماً از design.tyyi.net import کنید:

<link rel="stylesheet"
  href="https://design.tyyi.net/style.css">

حالت تاریک

سوییچ به <header> اضافه کنید:

<label><input type="checkbox"
  role="switch"> حالت تاریک</label>

ساختار HTML پایه

<html lang="fa" dir="rtl">
  <head>
    <link rel="stylesheet"
      href="style.css">
  </head>
  <body>
    <header><h1>...</h1></header>
    <main><section>...</section></main>
    <footer>...</footer>
  </body>
</html>

شخصی‌سازی

بازنویسی متغیرهای CSS در پروژه:

:root {
  --color-primary: #your-color;
  --radius: 8px;
}
